# Content Management System (CMS)

Internship Domain: Full-Stack Development

## Project Description
This project is a full-stack Content Management System (CMS) that allows
admin users to create, update, delete, and manage website content efficiently.

## Features
- Admin login
- Content creation and management
- Backend API using Node.js and Express
- Frontend using HTML, CSS, and JavaScript
- MongoDB database integration

## Technologies Used
- HTML, CSS, JavaScript
- Node.js
- Express.js
- MongoDB

## How to Run the Project
1. Install Node.js
2. Run `npm install`
3. Create `.env` file with MongoDB URL
4. Run `node server.js`
5. Open `index.html` in browser
